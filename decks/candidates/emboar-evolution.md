# emboar-evolution (Stage-2 ceiling)

Hypothesis: the pool's biggest sustained hitter is worth the setup
risk. Mega Emboar ex (932): 320 dmg for {R}{R}{C}, 380 HP — one-shots
everything in the pool and survives most answers.

- Line: 4× Tepig (567 — the 70 HP print, so **Buddy-Buddy Poffin can
  bench it directly**), 3× Pignite (931), 3× Mega Emboar ex (932).
- 3× Bloodmoon Ursaluna ex (44): colorless backup attacker while the
  line sets up, closer late.
- Consistency chain: 4 Poffin (turn-1 bench fill), 4 Mega Signal
  (finds M-Emboar), Hyper Aroma as Ace Spec (finds Pignite — the
  line's bottleneck), 4 Lacey + 3 Billy & O'Nare draw, 2 Boss's
  Orders, 3 Switch, 3 Potion, 23 {R} energy.

Risks: only 7 Basics (Poffin mitigates but a slow start loses to
aggro); two evolution steps ≈ 2 turns behind big basics; Crimson
Blast's 60 self-damage plus the 3-prize Mega rule-box makes trades
unforgiving. This candidate exists to measure whether the F3
(evolve) feature + search can actually pilot a setup deck — a
negative result is informative for H4.
